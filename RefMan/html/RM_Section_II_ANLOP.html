AN - Multifile Annotation Interface

ANannlen/afannlen -- Returns the length of an annotation.

ANannlist/afannlist -- Retrieves the annotation identifiers of an object.

ANatype2tag/afatypetag -- Returns the annotation tag corresponding to an annotation type.

ANcreate/afcreate -- Creates a data annotation for an object.

ANcreatef/affcreate -- Creates a file annotation.

ANend/afend -- Terminates access to an AN interface.

ANendaccess/afendaccess -- Terminates access to an annotation.

ANfileinfo/affileinfo -- Retrieves the number of annotations of each type in a file.

ANget_tagref/afgettagref -- Retrieves the tag/reference number pair of an annotation given its index and type.

ANid2tagref/afidtagref -- Retrieves the tag/reference number pair of an annotation given its identifier.

ANnumann/afnumann -- Returns the number of annotations of a given type attached to an object.

ANreadann/afreadann -- Reads an annotation.

ANselect/afselect -- Obtains an existing annotation.

ANstart/afstart -- Initializes the AN interface.

ANtag2atype/aftagatype -- Returns the annotation type corresponding to an annotation tag.

ANtagref2id/aftagrefid -- Returns the identifier of an annotation given its tag/reference number pair.

ANwriteann/afwriteann -- Writes an annotation.
