VSQ - Vdata Query Interface

VSQuerycount/vsqfnelt -- Retrieves the number of records in a vdata.

VSQueryfields/vsqfflds -- Retrieves the names of the fields in a vdata.

VSQueryinterlace/vsqfintr -- Retrieves the interlace mode of the vdata.

VSQueryname/vsqfname -- Retrieves the name of a vdata.

VSQueryref/vsqref -- Returns the reference number of a vdata.

VSQuerytag/vsqtag -- Returns the tag of the specified vdata.

VSQueryvsize/vsqfvsiz -- Retrieves the size of a record in a vdata.
