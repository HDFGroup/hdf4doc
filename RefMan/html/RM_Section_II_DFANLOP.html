DFAN - Single-file Annotation Interface

Note: This interface is abc deprecated.

DFANaddfds/daafds -- Adds a file description to a file.

DFANaddfid/daafid -- Writes a file label to a file.

DFANclear/daclear -- Resets all internal library structures and parameters of the DFAN annotation interface.

DFANgetdesc/dagdesc -- Reads the description assigned to the data object with the given tag and reference number.

DFANgetdesclen/dagdlen -- Retrieves the length of a description of the data object with the given tag and reference number.

DFANgetfds/dagfds -- Reads the next file description.

DFANgetfdslen/dagfdsl -- Returns the length of a file description.

DFANgetfid/dagfid -- Reads a file label from a file.

DFANgetfidlen/dagfidl -- Returns the length of a file label.

DFANgetlabel/daglab -- Reads the label assigned to the data object identified by the given tag and reference number.

DFANgetlablen/dagllen -- Returns the length of a label assigned to the object with a given tag and reference number.

DFANlablist/dallist -- Returns a list of all reference numbers and labels (if labels exist) for a given tag.

DFANlastref/dalref -- Returns the reference number of the annotation last written or read.

DFANputdesc/dapdesc -- Writes a description for the data object with the given tag and reference number.

DFANputlabel/daplab -- Assigns a label to the data object with the given tag/reference number pair.
