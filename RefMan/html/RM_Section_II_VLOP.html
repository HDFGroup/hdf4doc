V -- Vgroup Interface

Vaddtagref/vfadtr -- Inserts an object into a vgroup.

Vattach/vfatch -- Initiates access to a new or existing vgroup.

Vattrinfo/vfainfo -- Retrieves the name, data type, number of values, and value size of an attribute assigned to a vgroup.

Vattrinfo2 -- Retrieves information of an attribute assigned to a vgroup (either new or old style attribute.)

Vdelete/vdelete -- Remove a vgroup from a file.

Vdeletetagref/vfdtr -- Deletes an object from a vgroup.

Vdetach/vfdtch -- Terminates access to a vgroup.

Vend/vfend -- Terminates access to a vgroup and/or vdata interface.

Vfind/vfind -- Returns the reference number of a vgroup given its name.

Vfindattr/vffdatt -- Returns the index of a vgroup attribute given its name.

Vfindclass/vfndcls -- Returns the reference number of a vgroup specified by its class name.

Vflocate/vffloc -- Locates a vdata in a vgroup given a list of field names.

Vgetattdatainfo -- Retrieves location and size of attribute's data.

Vgetattr/vfgnatt/vfgcatt -- Retrieves the values of a vgroup attribute.

Vgetattr2 -- Retrieves the values of a vgroup attribute (either new or old style.)

Vgetclass/vfgcls -- Retrieves the class name of a vgroup.

Vgetclassnamelen -- Retrieves the length of a vgroup's class name.

Vgetid/vfgid -- Returns the reference number of the next vgroup.

Vgetname/vfgnam -- Retrieves the name of a vgroup.

Vgetnamelen -- Retrieves the length of a vgroup's name.

Vgetnext/vfgnxt -- Gets the reference number of the next member (vgroup or vdata only) of a vgroup.

Vgettagref/vfgttr -- Retrieves the tag/reference number pair of an object given its index within a vgroup.

Vgettagrefs/vfgttrs -- Retrieves the tag/reference number pairs of the HDF objects belonging to a vgroup.

Vgetversion/vfgver -- Gets the version of a vgroup.

Vgetvgroups/vfgvgroups -- Retrieves reference numbers of vgroups in a file or in a vgroup.

Vgisinternal -- Determine if a vgroup was created by the library internally.

Vinqtagref/vfinqtr -- Checks whether an object belongs to a vgroup.

Vinquire/vfinq -- Retrieves the number of entries in a vgroup and its name.

Vinsert/vfinsrt -- Inserts a vdata or vgroup into a vgroup.

Visvg/vfisvg -- Determines whether an element of a vgroup is a vgroup and a member of another vgroup.

Visvs/vfisvs -- Determines whether a data object is a vdata within a vgroup.

Vlone/vflone -- Retrieves the reference numbers of lone vgroups, i.e., vgroups that are at the top of the grouping hierarchy, in a file.

Vnattrs/vfnatts -- Returns the number of attributes assigned to a vgroup.

Vnattrs2 -- Returns the number of new- and old-style attributes assigned to a vgroup.

Vnrefs/vnrefs -- Returns the number of tags of a given tag type in a vgroup.

Vntagrefs/vfntr -- Returns the number of objects in a vgroup.

Vsetattr/vfsnatt/vfscatt -- Attaches an attribute to a vgroup.

Vsetclass/vfscls -- Sets the class name of a vgroup.

Vsetname/vfsnam -- Sets the name of a vgroup.

Vstart/vfstart -- Initializes the vdata and/or vgroup interface.

VHmakegroup/vhfmkgp -- Creates a vgroup.

VQueryref/vqref -- Returns the reference number of a vgroup.

VQuerytag/vqtag -- Returns the tag of a vgroup.
